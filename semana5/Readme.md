Durante esta semana 5 vimos la instrucción "for" que nos ayuda a crear un bucle de repeticiones definidas.

Con esta función se realizaron 3 ejemplos 

Realizamos 4 ejercicios:

Ejercicio 1: Se realizó un programa que determinara el exponencial, logaritno, seno, coseno y raíz cuadrada de un número dado por el usuario.
Este contiene un intervalo y un espaciado también establecido por el usuario. Con la ayuda de la función for se pudo establecer el intervalo.

Ejercicio 2: Se creó un programa que calculara el factorial de un número dado por el usuario. El ejercicio fue mas sencillo gracias a la función fac.

Ejercicio 3: En este ejercicio se hizo un programa que calculara los números primos de un intervalo definido por el usuario.
Con ayuda de la criba de eratostenes se calcularon los números primos 

Ejercicio 4: Este fue el más complicado. Se realizó un programa que convirtiera números binarios a decimales y viceversa.
Fue de vital importancia conocer la manera en que se convierten estos números.
